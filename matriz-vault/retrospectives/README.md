# Retrospectivas

Reflexión al final de cada ciclo. Formato `Retro — [nombre del ciclo].md`.

Usar template: `tpl-retrospectiva`

## Cadencia sugerida

- Cada 1-2 semanas (alineado con ciclos de Linear)
- O al cerrar un milestone importante

## Foco

- Qué salió bien (para repetir)
- Qué podría mejorar (para ajustar)
- Máximo 3 acciones concretas (para no sobrecargar)
